# Book-Store-Management-SQL-Project
SQL-based Online Bookstore Management System for analyzing customer orders, book sales, revenue, and inventory management.

# Project Overview

This project demonstrates the design and analysis of an Online Bookstore database using SQL. The database manages information related to books, customers, and orders. Various SQL queries were used to perform business analysis, sales reporting, customer analysis, and inventory management.

# Database Schema

use onlineBookstore;

The project consists of three main tables:
1)Customers
2)Books
3)Orders

# Key Analysis Performed
1)Retrieve books by genre and publication year
2)Analyze customer orders and purchasing behavior
3)Calculate total revenue generated from book sales
4)Identify top-selling and most frequently ordered books
5)Find customers with multiple orders
6)Analyze book sales by genre and author
7)Determine highest-spending customers
8)Monitor inventory and remaining stock after sales
9)Generate business insights using SQL queries

# Project Outcome
This project demonstrates practical SQL skills in database design, data analysis, sales reporting, customer analytics, and inventory management. It showcases the ability to work with relational databases and extract meaningful business insights using SQL queries.
